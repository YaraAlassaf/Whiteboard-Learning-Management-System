# Whiteboard Learning Management System

## Project Overview

This project presents a project management plan for an enhanced Whiteboard Learning Management System (LMS). The proposed solution focuses on improving communication between students, instructors, and administrators by introducing an independent real-time academic notification system.

The project applies Project Management principles, including project planning, requirements analysis, Work Breakdown Structure (WBS), Gantt Chart, RACI Matrix, Risk Register, and Scope Management.

## Project Objectives

- Improve academic communication through real-time notifications.
- Reduce missed assignments, exams, and important announcements.
- Support multi-channel notifications (Dashboard, Email, and SMS).
- Enhance system usability and overall user experience.
- Improve synchronization across different platforms and devices.

## Project Features

- Real-time academic notifications
- Multi-channel alert delivery
- Role-based access control
- Customizable notification preferences
- Improved system performance
- User-friendly interface
- Project Charter
- Scope Statement
- WBS & WBS Dictionary
- Gantt Chart
- RACI Matrix
- Risk Register & Risk Matrix

## Development Methodology

The project follows the Incremental Model, allowing the system to be developed and tested in multiple functional increments while reducing development risks and incorporating user feedback throughout the project lifecycle.
